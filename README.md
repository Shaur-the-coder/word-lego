An interactive web application for building creative words from modular components (prefixes, roots, suffixes, chaos modifiers, and connectors). Built with React and drag-and-drop functionality.

## Features

- **Drag & Drop Interface**: Build words by dragging blocks into the word builder
- **Rich Block Library**: 70+ prefixes, roots, suffixes, chaos modifiers, and connectors
- **Search Functionality**: Real-time search across all blocks
- **Custom Blocks**: Add your own word components
- **AI-Simulated Definitions**: Hover over words for contextual definitions
- **Trash Can**: Drag blocks to delete them
- **Randomize**: Generate random word combinations instantly

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool and dev server
- **dnd-kit** - Drag-and-drop library
- **CSS** - Lego-inspired styling

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

## Usage

1. **Search** for blocks using the search bar
2. **Drag** blocks from the panel to the word builder
3. **Reorder** blocks by dragging within the builder
4. **Delete** blocks by dragging to the trash can
5. **Hover** over your word to see its definition
6. **Randomize** to generate new combinations
7. **Add Custom** blocks for personalized word building
